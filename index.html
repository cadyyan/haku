    A Minecraft mod for adding some new magic creatures

    Project maintained by cadyyan Hosted on GitHub Pages — Theme by mattgraham

    What is it?

    Haku is a mod that adds some different magical creatures that are inspired from some anime sources. For now the creatures are brought in for creating some fun maps but in the feature we hope to make them part of survival Minecraft.

    Creatures

    Many magical creatures are added or will be added by this mod. Each is given with a short description and a picture (if available) as well as the development status.

    • Susuwatari

      Also known as soot spirits, these creatures are made of soot that has been animated. They are strong workers that can be made to perform basic work such as moving coal to a furnace. They love eating star foods.
